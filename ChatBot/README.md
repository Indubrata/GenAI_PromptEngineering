# Simple AI Tool

A lightweight, terminal-based AI tool built in Python. It takes user input, sends it to the Google Gemini API, and displays the response.

## Features

1. **Explain a Concept**: Explains a concept clearly and concisely.
2. **Summarize Text**: Provides a comprehensive summary of any provided text.
3. **Explain Like I'm 5 (ELI5)**: Breaks down complex topics into simple terms as if speaking to a 5-year-old child.

## Prerequisites

- Python 3.7+
- A Google Gemini API Key. You can get one from [Google AI Studio](https://aistudio.google.com/).

## Installation

Install the required Python library:

pip install google-generativeai
   
## Usage

Run the script from your terminal:

python ai_tool.py

Follow the on-screen prompts to:
1. Provide your API key (if not set in the environment).
2. Select a feature from the menu.
3. Enter the text or concept you want the AI to process.
4. View the result!
